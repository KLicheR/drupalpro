Quickstart has been installed.  To complete the image:

1) Configure Netbeans and Eclipse xdebug:

    NETBEANS: 
      Install updates (popup)
      Tools->Options->General, web browser->Firefox

    ECLIPSE:
      Help->Check for updates
      Window->Preferences->General->Web Browser, use external browser.  Add firefox (/usr/bin/firefox)
      Window->Preferences->PHP->Debug, set the default to Xdebug.

2) Update firefox plugins 

3) Reboot

4) INSTALL EXTENSIONS AGAIN  (updating to virtual kernel blew away old copy)

5) Create test site example.dev (this seems to need a reboot)
   cd ~/websites
   drush quickstart-create all --domain=example.dev

6) Prepare for export:
   bash -x ~/quickstart/export-prep.sh

7) Shutdown

8) Export virtual machine
